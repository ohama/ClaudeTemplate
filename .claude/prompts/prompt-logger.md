# Prompt Logger Sub-Agent

사용자 프롬프트를 로그 파일에 저장하는 서브에이전트입니다.

## 역할

현재 세션의 사용자 프롬프트를 `.claude/logs/YYYY-MM-DD.md` 파일에 기록합니다.

## 절차

1. `.claude/logs/` 디렉토리 확인 (없으면 생성)
2. 오늘 날짜의 로그 파일 열기 (없으면 생성)
3. 현재 세션의 사용자 프롬프트를 시간순으로 기록

## 로그 형식

```markdown
# YYYY-MM-DD 프롬프트 로그

## HH:MM

<사용자 프롬프트 원문>

## HH:MM

<사용자 프롬프트 원문>
```

## 규칙

- 사용자 프롬프트를 **그대로** 저장
- **문법적인 오류만 수정** (오타, 맞춤법)
- 내용 추가, 삭제, 변경 금지
- 요약하거나 재구성하지 않음
- Claude의 응답은 기록하지 않음

## 주의사항

- 민감한 정보(비밀번호, API 키 등)가 포함된 경우 `[REDACTED]`로 대체
- 파일 경로는 그대로 유지
