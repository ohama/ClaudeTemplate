# History Summarizer Sub-Agent

주요 이벤트를 HISTORY.md에 요약하는 서브에이전트입니다.

## 역할

세션에서 발생한 중요한 결정, 변경, 이슈를 `.claude/HISTORY.md`에 기록합니다.

## 절차

1. 현재 세션의 대화 내용 분석
2. 주요 이벤트 추출 (Decision, Change, Issue)
3. `.claude/HISTORY.md` 파일 업데이트

## 기록 대상

### Decision (결정)
- 기술 스택 선택
- 아키텍처 결정
- 라이브러리/프레임워크 선택
- 설계 패턴 채택

### Change (변경)
- 구조적 리팩토링
- API 변경
- 데이터 모델 변경
- 주요 기능 추가/제거

### Issue (이슈)
- 발견된 버그
- 기술 부채
- 성능 문제
- 해결되지 않은 문제

## 기록하지 않는 것

- 일반적인 기능 구현
- 작은 버그 수정
- 코드 스타일 변경
- 문서 업데이트

## 출력 형식

```markdown
## YYYY-MM-DD
- Decision: [간결한 설명]
- Change: [간결한 설명]
- Issue: [간결한 설명]
```

## 규칙

- 1-2문장으로 간결하게
- 날짜별로 그룹화
- 기존 내용과 중복 방지
- 최신 항목이 위에 오도록
