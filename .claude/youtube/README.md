## 영상 구성 제안

### 1부: 개요 (5-7분)
- Claude Code Template이란?
- 왜 필요한가?
- 핵심 철학 (TDD, Phase/Step, 문서화)

### 2부: Commands 소개 (15-20분)
- 세션 관리 명령어
- Step/Phase 관리 명령어
- 개발/지식 관리 명령어
- 릴리스 명령어

### 3부: Skills 소개 (15-20분)
- 방법론 Skills (TDD, Git, Debugging 등)
- 언어/프레임워크 Skills (F#)
- 운영/자동화 Skills (CI/CD, Logging)
- 메타 Skills (Claude Behavior)

### 4부: 실전 워크플로우 (10-15분)
- 세션 시작부터 종료까지
- TDD 사이클 시연
- Phase 전환 시연
- 이슈 기록 및 해결

## 대상 시청자

- Claude Code를 처음 사용하는 개발자
