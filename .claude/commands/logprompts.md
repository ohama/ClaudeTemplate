# logprompts

현재 세션의 사용자 프롬프트를 로그 파일에 저장합니다.

## 로그 위치

`.claude/logs/YYYY-MM-DD.md`

## 규칙

- 사용자가 입력한 프롬프트를 **그대로** 저장
- **문법적인 오류만 수정** (오타, 맞춤법)
- 내용 추가, 삭제, 변경 금지
- 요약하거나 재구성하지 않음

## 실행

1. `.claude/logs/` 디렉토리가 없으면 생성
2. 오늘 날짜의 로그 파일 열기 (없으면 생성)
3. 현재 세션의 사용자 프롬프트를 시간순으로 기록
4. 타임스탬프와 함께 추가

## 로그 형식

```markdown
## HH:MM

<사용자 프롬프트 원문 (문법 수정만)>
```
